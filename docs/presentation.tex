\begin{frame}
  \frametitle{Defense-Prep Corrections}
  The defense-readiness pass focused on closing the gap between claims and reality.

  \begin{itemize}
    \item Repaired config parsing for scalar YAML values and direct device/checkpoint fields
    \item Fixed \texttt{BaseTrainer} to use the typed config object correctly
    \item Stabilized predictive entropy to respect theoretical bounds
    \item Made OOD thresholding deterministic and data-fit rather than query-fit
    \item Added real Grad-CAM support to the inference API
    \item Began integrating the custom exception hierarchy into runtime code
    \item Reworked the pytest harness to avoid inaccessible temp-directory failures
  \end{itemize}
\end{frame}
