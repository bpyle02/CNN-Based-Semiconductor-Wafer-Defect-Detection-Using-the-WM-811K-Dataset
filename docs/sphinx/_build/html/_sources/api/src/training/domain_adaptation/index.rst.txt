src.training.domain_adaptation
==============================

.. py:module:: src.training.domain_adaptation

.. autoapi-nested-parse::

   Domain Adaptation for cross-plant wafer defect detection.

   Implements techniques to adapt models trained on one wafer plant
   to generalize to different plants with different defect characteristics.

   .. method:: - Fine-tuning

      Simple transfer learning on target domain

   .. method:: - CORAL

      Correlation Alignment for domain-invariant features

   .. method:: - Domain-adversarial training

      Adversarial loss to align source and target
      

   .. rubric:: References

   [40] Sun & Saenko (2016). "Deep CORAL". arXiv:1607.01719
   [41] Ganin et al. (2016). "Domain-Adversarial Training". arXiv:1505.07818
