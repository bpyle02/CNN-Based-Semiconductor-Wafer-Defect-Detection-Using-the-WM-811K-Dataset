.. py:class:: TrainConfig

   Training configuration for wafer defect detection models.

   .. attribute:: num_epochs

      Number of training epochs (default: 5 for CPU feasibility)

   .. attribute:: batch_size

      Mini-batch size (default: 64)

   .. attribute:: learning_rate

      Initial learning rate (default: 1e-3 for CNN, 1e-4 for pretrained)

   .. attribute:: weight_decay

      L2 regularization coefficient (default: 1e-4)

   .. attribute:: num_classes

      Number of output classes (default: 9)

   .. attribute:: seed

      Random seed for reproducibility (default: 42)

   .. attribute:: device

      Compute device ('cuda' or 'cpu')

   .. attribute:: scheduler_patience

      Patience for ReduceLROnPlateau (default: 3)

   .. attribute:: scheduler_factor

      LR reduction factor (default: 0.5)

   .. attribute:: loss_weights

      Per-class weights for CrossEntropyLoss (set during data prep)
