.. py:function:: balance_dataset_with_synthetic(wafer_maps: numpy.ndarray, labels: numpy.ndarray, target_per_class: Optional[int] = None, size: int = 96, max_target_per_class: int = 10000) -> Tuple[numpy.ndarray, numpy.ndarray]

   Balance a dataset by generating synthetic samples for underrepresented classes.

   :param wafer_maps: Array of preprocessed (H, W) wafer maps
   :param labels: 1D array of integer class labels
   :param target_per_class: Target count per class (None = min(max_class, max_target_per_class))
   :param size: Size of generated maps
   :param max_target_per_class: Safety cap when target_per_class is None. Keeps memory
                                bounded when the majority class dwarfs rare ones (WM-811K "none" has ~147k;
                                matching it across 9 classes would allocate ~40 GB).

   :returns: Tuple of (augmented_maps, augmented_labels)
