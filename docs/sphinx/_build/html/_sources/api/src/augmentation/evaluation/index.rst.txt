.. py:class:: FIDScorer

   Fréchet Inception Distance (FID) scorer.

   Measures distributional similarity between real and synthetic data
   using Inception network features.


   .. py:method:: compute_fid(real_features: numpy.ndarray, synthetic_features: numpy.ndarray) -> float
      :staticmethod:


      Compute FID between real and synthetic features.

      :param real_features: Real sample features (n_real, feature_dim)
      :param synthetic_features: Synthetic sample features (n_synthetic, feature_dim)

      :returns: FID score (lower is better, 0 is perfect)
