src.models.cnn
==============

.. py:module:: src.models.cnn

.. autoapi-nested-parse::

   Custom CNN architecture for wafer defect classification.

   Implements a lightweight convolutional neural network optimized for CPU inference
   on 96x96 wafer map images.

   Spatial Pyramid Pooling layer.

   Pools at multiple spatial scales and concatenates results,
   preserving multi-scale spatial information that GAP discards.

   :param pool_sizes: List of spatial output sizes for each pooling level.
                      Default (1, 2, 4) gives 1x1 + 2x2 + 4x4 = 21 spatial bins.

   Initialize internal Module state, shared by both nn.Module and ScriptModule.

.. py:class:: WaferCNN(num_classes: int = 9, input_channels: int = 3, feature_channels: Sequence[int] | None = (32, 64, 128, 256), dropout_rate: float = 0.5, head_hidden_dim: int | None = 128, head_dropout: float | None = 0.3, use_batch_norm: bool = True, use_spp: bool = False, spp_pool_sizes: Sequence[int] | None = (1, 2, 4))

   Bases: :py:obj:`torch.nn.Module`


   Custom CNN for semiconductor wafer defect classification.

   Architecture:
       - Input: 3x96x96 (replicated grayscale channels)
       - Configurable convolutional feature stages
       - Global average pooling
       - Configurable classifier head depth and dropout
       - Output: logits for ``num_classes``

   The defaults preserve the existing architecture, but the feature widths and
   classifier head can now be adjusted from training code without changing the
   model definition itself.

   Initialize the custom CNN.

   :param num_classes: Number of output classes (default 9 for wafer defects)
   :param input_channels: Number of input channels. Default keeps the current
                          replicated grayscale convention.
   :param feature_channels: Output width for each convolutional stage.
   :param dropout_rate: Dropout probability before the classifier head.
   :param head_hidden_dim: Optional hidden width for the classifier head. Set
                           to ``None`` for a single linear layer.
   :param head_dropout: Dropout probability between hidden and output layers
                        when ``head_hidden_dim`` is enabled.
   :param use_batch_norm: Whether to apply BatchNorm after each convolution.
