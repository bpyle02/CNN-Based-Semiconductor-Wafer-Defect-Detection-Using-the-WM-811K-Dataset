src.models.vit
==============

.. py:module:: src.models.vit

.. autoapi-nested-parse::

   Vision Transformer (ViT) architecture for wafer defect classification.

   Adapts the Vision Transformer from Dosovitskiy et al. to work with 96x96 wafer maps.
   Includes patch embedding, transformer encoder, and classification head.

   Convert image to patch embeddings.

   Splits input image into non-overlapping patches and linearly embeds them.
   Positional encodings are added by the ViT class after CLS token prepend.

   :param image_size: Height/width of input image (assumed square)
   :param patch_size: Height/width of each patch
   :param in_channels: Number of input channels (e.g., 3 for RGB)
   :param embed_dim: Dimension of patch embeddings

.. py:class:: ViT(image_size: int = 96, patch_size: int = 8, in_channels: int = 3, num_classes: int = 9, embed_dim: int = 384, num_heads: int = 6, num_layers: int = 12, mlp_dim: int = 1536, dropout: float = 0.1, attention_dropout: float = 0.0)

   Bases: :py:obj:`torch.nn.Module`


   Vision Transformer for image classification.

   Adapts ViT from Dosovitskii et al. (An Image is Worth 16x16 Words, ICLR 2021)
   to work with 96x96 wafer maps.

   Architecture:
       - Patch embedding (96x96 -> 12x12 = 144 patches of 8x8)
       - Class token prepended
       - Positional embeddings added
       - Transformer encoder stack
       - Classification head (MLP on [CLS] token)

   :param image_size: Input image size (96)
   :param patch_size: Patch size (16)
   :param in_channels: Input channels (3)
   :param num_classes: Number of classes (9)
   :param embed_dim: Embedding dimension (384 for ViT-small)
   :param num_heads: Number of attention heads (6 for ViT-small)
   :param num_layers: Number of transformer layers (12)
   :param mlp_dim: Feed-forward dimension
   :param dropout: Dropout rate
   :param attention_dropout: Attention dropout rate

   Initialize internal Module state, shared by both nn.Module and ScriptModule.
