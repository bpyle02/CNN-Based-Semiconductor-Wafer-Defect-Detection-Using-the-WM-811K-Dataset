src.inference.server

FastAPI-based real-time inference server for wafer defect classification.

Provides REST endpoints for model serving with support for: - Image-based predictions (base64 or file upload) - Multi-model serving with dynamic loading and model registry - Health checks and model introspection - Comprehensive request validation and error handling - CORS support for web clients - Performance metrics tracking
