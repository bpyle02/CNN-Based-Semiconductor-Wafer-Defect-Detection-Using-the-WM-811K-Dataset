src.inference.calibration

Temperature scaling for post-hoc probability calibration.

Fits a single scalar T on a held-out validation set by minimizing negative log-likelihood of softmax(logits / T). Scaling logits by a learned temperature does not change argmax predictions but it does repair over/under-confident softmax outputs, which matters for ECE, threshold-based decisioning, and downstream ensembling.

Reference:

Guo, Pleiss, Sun, Weinberger (2017). “On Calibration of Modern Neural Networks.” ICML. arXiv:1706.04599

class src.inference.calibration.TemperatureScaling[source]

Post-hoc calibration by scaling logits by learned temperature T.

Fit T by minimizing NLL on a held-out validation set. T > 1 softens, T < 1 sharpens. Does not change argmax predictions but makes softmax probabilities trustworthy.
