src.utils.seeding

Centralized seed management.

A single entry point (seed_everything()) that sets seeds across every RNG source the project touches: Python’s random module, NumPy, PyTorch (CPU and CUDA), and the PYTHONHASHSEED environment variable.

Idempotent: calling twice with the same seed produces the same state.

src.utils.seeding.seed_everything(seed: int = 42) → None[source]

Set seeds for python random, numpy, torch (CPU + CUDA), and PYTHONHASHSEED.

Logs the seed at INFO. Idempotent. Returns nothing.

Parameters:

seed – Seed value; must fit in an unsigned 32-bit integer (0 <= seed <= 2**32 - 1). Defaults to 42.

Raises:

ValueError – If seed is outside the valid range.
