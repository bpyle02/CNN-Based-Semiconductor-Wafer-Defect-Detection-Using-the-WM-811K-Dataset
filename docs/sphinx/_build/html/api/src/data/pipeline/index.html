Load, split, and optionally balance the WM-811K dataset.

Uses the pre-resized NPZ cache produced by scripts/precompute_tensors.py when present (memory-mapped to keep RSS bounded on Colab), otherwise falls back to loading the raw pickle.

Returns a dict with keys: train_maps, y_train, val_maps, y_val, test_maps, y_test, loss_weights, class_names.
