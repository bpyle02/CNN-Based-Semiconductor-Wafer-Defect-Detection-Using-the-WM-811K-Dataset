Byzantine-robust aggregation methods for federated learning.

Implements robust aggregation strategies that can tolerate malicious or faulty clients in federated learning settings.

class src.federated.aggregation.Krum[source]

Krum aggregation: selects parameter vector closest to most others.

Filters out potential Byzantine updates by selecting the gradient vector that is closest (in Euclidean distance) to the maximum number of other gradient vectors. Robust to f malicious clients where n >= 2f + 3 (per Blanchard et al., 2017).

class src.federated.aggregation.MultiKrum[source]

Multi-Krum: selects m parameter vectors via repeated Krum.

Applies Krum m times, removing the selected vector each iteration. More robust than single Krum as it combines multiple diverse updates.
