src.training.simclr

SimCLR: A Simple Framework for Contrastive Learning of Visual Representations.

Implements SimCLR (Chen et al., ICML 2020) for self-supervised pretraining on unlabeled wafer maps. Learns useful representations without labels by maximizing agreement between different augmentations.
